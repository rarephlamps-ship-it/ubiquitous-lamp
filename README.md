# Ondskabens Skov

Native Android survival-horror om Line, Jean og det levende hus i skoven.

Projektet indeholder touch-styring, fem zoner, sanity, batteri, lommelygte, tændstikker, tre symboler, dagbøger, flere fjendetyper, bosskamp, procedural grafik, syntetisk lydlandskab, automatisk save og flere slutninger.

Den installerbare APK bygges automatisk med GitHub Actions.
